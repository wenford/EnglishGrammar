# 并列人称代词的排列顺序

1) 单数人称代词并列作主语时，其顺序为：  

```
　　　 第二人称 －> 第三人称 －> 第一人称
　　　　　　you －> he/she;　it -> I
　　　 You, he and I should return on time.  

```
　2) 复数人称代词作主语时，其顺序为：

```
　　　 第一人称 －> 第二人称 －> 第三人称
　　　　　　we　－>　you　　 －>　They
```
***注意***： 在下列情况中，第一人称放在前面。    

　　　a. 在承认错误，承担责任时,

```
　　　It was I and John that made her angry.
　　　是我和约翰惹她生气了。
```

　　　b. 在长辈对晚辈，长官对下属说话时，如长官为第一人称， 如：I and you try to finish it.

　　　c. 并列主语只有第一人称和第三人称时，

　　　d. 当其他人称代词或名词被定语从句修饰时。
